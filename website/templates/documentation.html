YourAnalyst Documentation

Your personal data analysis companion

Introduction

YourAnalyst is a powerful web-based tool designed to help you analyze, clean, transform, and visualize datasets with ease.

Features

  • Data Cleaning: Drop nulls, fill missing values, remove duplicates.
  • Data Transformation: Normalize, Standardize, Encode Categorical, PCA, Clustering.
  • Statistical Analysis: Basic Stats, Outlier Detection, Covariance & Correlation.
  • Visualizations: Histogram, Scatter Plot, Bar Plot, Line Plot (with dashboard export).
  • Column Type Conversion: Easily change data types (int, float, string).
  • Data Export: Download modified dataset as CSV & export dashboard as HTML.

How to Use

  1. Upload your dataset on the upload page.
  2. Select a table if there are multiple.
  3. Click on Analyse button.
  4. Download your AI-Generated(It takes a little time) as readme file.
  5. Go to Manual Analysis.
  6. Select columns and choose an operation to perform.
  7. Visualize your data or download results.

Tips

  • Ensure correct column selection for each operation.
  • For charts, selecting one or two columns gives different results.
  • Use Undo to revert the last action if needed.
  • Dashboard allows saving & exporting multiple charts together.
